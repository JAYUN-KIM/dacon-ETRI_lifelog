## 프로젝트 개요

이 프로젝트는 라이프로그 데이터를 기반으로 사용자의 **수면 상태**, **피로도**, **스트레스**를 예측하는  
**제5회 ETRI 휴먼이해 AI 논문경진대회** 참가 기록 및 실험 저장소입니다.

- **대회명**: 제5회 ETRI 휴먼이해 AI 논문경진대회
- **플랫폼**: DACON
- **목표**: 라이프로그 기반 다중 타겟 예측
- **평가 지표**: Average Log-Loss
- **최종 산출물**:
  - 대회 제출용 예측 결과
  - 실험 로그 및 재현 가능한 코드
  - ICTC 2026 논문 초안

---

| 타겟 | 문제 유형 | 설명 |

| Q2 | 이진 분류 | 취침 전 피로도 |
